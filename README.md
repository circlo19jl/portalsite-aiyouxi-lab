# portalsite-aiyouxi-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立开发的 HTML 页面资源。这些页面通常以单文件或少量依赖的形式存在，适合直接通过浏览器打开查看。仓库本身不针对特定网站或域名，仅作为页面归档与版本管理的工具。

## 目录说明

```
├── pages/          # 存放各个独立 HTML 页面及其相关资源
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- **pages/**: 每个子目录或文件对应一个独立 HTML 页面，可能包含配套的样式、脚本等。
- **assets/**: 若页面间存在共享资源，可放置于此。

## 页面归档说明

- 每个页面均为独立开发，不依赖外部服务或数据库。
- 页面可能包含演示、工具、原型等用途，具体功能请参考页面内说明。
- 部分页面可能需要现代浏览器支持，建议使用最新版 Chrome、Firefox 或 Edge 访问。

## 维护说明

- 本仓库不定期更新，新增或修改页面会记录在 Git 历史中。
- 欢迎通过 Issue 报告问题或提出建议。
- 如需贡献页面，请提交 Pull Request，并确保页面内容不包含敏感信息或外部依赖。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/yourusername/portalsite-aiyouxi-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
